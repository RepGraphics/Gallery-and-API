API Documentation

This document provides an overview of the available API endpoints and their usage. All API responses are in JSON format.

API Base Endpoint:

https://api.yourdomain.uk

API Status

GET /api/status

Returns the status and version of the API.

Authorization

Each endpoint requires a valid token in the Authorization header in the format:

Authorization: Bearer <your_token_here>

Note: Endpoints marked as Developer Only Access! require special privileges and can only be accessed by authorized developers.
